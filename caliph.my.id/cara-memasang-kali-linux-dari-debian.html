Cara memasang Kali Linux dari Debian

Kali Linux adalah distribusi Linux populer yang didasarkan pada Debian yang dikenal dengan kekuatan alat-alat uji penetrasi dan keamanan. Dalam tutorial ini, kami akan memandu Anda melalui proses membangun Kali Linux dari instalasi Debian. Dengan mengubah sumber instalasi apt dan menginstal paket-paket yang diperlukan, Anda dapat mengubah sistem Debian Anda menjadi Kali Linux.

 

Mengubah Sumber Apt

Langkah pertama adalah mengubah isi file /etc/apt/sources.list. Buka terminal dan jalankan perintah berikut:

sudo nano /etc/apt/sources.list

Ganti kontennya dengan baris berikut:

deb http://kali.download/kali kali-rolling main non-free contrib

Simpan perubahan dengan menekan Ctrl + X, kemudian Y untuk mengonfirmasi.

Mendapatkan Kunci Apt Kali

Selanjutnya, kita perlu mendapatkan kunci apt Kali untuk mengotentikasi sumber apt Kali Linux. Ada tiga metode untuk melakukannya:

Metode 1: Unduh Kunci ASC Langsung dengan Wget

Jika Anda memiliki wget terpasang, Anda dapat menggunakan perintah berikut untuk mengunduh kunci:

sudo wget https://archive.kali.org/archive-key.asc -O /etc/apt/trusted.gpg.d/kali-archive-keyring.asc

Metode 2: Apt-key (Menggunakan apt-key tidak direkomendasikan)

Alternatifnya, Anda dapat menggunakan apt-key untuk mengunduh kunci:

sudo apt update && sudo apt install -y wget curl gnupg2
sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys ED444FF07D8D0BF6

Metode 3: Pasang Paket Kunci Kali

Unduh paket Kali Archive Keyring dari situs web resmi Kali Linux dan pasang dengan perintah berikut:

wget https://www.kali.org/archive-key.asc
sudo apt-key add archive-key.asc

Memperbarui dan Memasang Paket Kali-Linux

Setelah Anda mendapatkan kunci, Anda dapat memperbarui sumber paket apt dan memasang paket-paket Kali-Linux:

sudo apt update
sudo apt upgrade
sudo apt install kali-defaults

Untuk menyesuaikan instalasi Kali Linux Anda, Anda dapat memilih toolset dan lingkungan desktop yang diinginkan. Gunakan perintah berikut untuk mencari dan mem asang paket-paket tertentu:

apt-cache search kali-tools
apt-cache search kali-linux
apt-cache search kali-desktop

Misalnya, untuk memasang Kali Toolset lengkap dalam XFCE4 dengan xrdp untuk akses jarak jauh, jalankan perintah berikut:

sudo apt install kali-linux-everything kali-desktop-xfce xrdp

Menambahkan Pengguna Non-Root

Disarankan untuk tidak menggunakan akun root untuk aktivitas sehari-hari. Sebagai gantinya, buat pengguna non-root dengan hak akses sudo:

sudo useradd -m -G sudo -s /bin/bash kaliuser
sudo passwd kaliuser

Perintah pertama menambahkan pengguna baru dengan nama "kaliuser" dengan hak akses sudo, membuat direktori home, dan menetapkan shell default ke /bin/bash. Perintah kedua memungkinkan Anda untuk mengatur kata sandi untuk pengguna baru.

Selamat! Anda telah berhasil memasang Kali Linux dari Debian. Selamat mengeksplorasi alat dan fitur canggih yang dimilikinya!
